# CST 文件命名规范

## 概述

统一项目内仿真模型和仿真结果的命名格式，便于识别和管理。

---

## 一、仿真模型命名

### 1.1 超表面模型

```
年月日_hex_metasurface_[阵面尺寸]_[功能]_[馈源位置]_序号.cst
```

#### 格式说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 年月日 | 6位日期 | 260416 |
| hex_metasurface | 项目标识（六边形超表面） | hex_metasurface |
| 阵面尺寸 | 方形：宽_高，圆形：r直径 | 180_180, r180 |
| 功能 | 用途描述 | colli, steer, multi |
| 馈源位置 | center / offcenter / center_mm | center |
| 序号 | **仅当同一日期同类文件有多个时才加** | 01, 02 |

#### 序号规则

- 喇叭文件：center 和 offcenter 视为不同类型，**不需序号**
- 超表面文件：仅当同一日期有多个相同参数时才加序号
- 不同阵面尺寸/功能视为不同类型，**不需序号**

#### 示例

```
260416_hex_metasurface_180_180_colli_center.cst        # 180×180方形阵面，准直，中心馈源
260416_hex_metasurface_180_180_steer_offcenter.cst     # 180×180方形阵面，波束扫描，偏心馈源
260416_hex_metasurface_r180_colli_center.cst           # 直径180圆形阵面，准直，中心馈源
260416_hex_metasurface_200_200_multi_center_01.cst    # 同日期有两个200×200阵面时
260416_hex_metasurface_200_200_multi_center_02.cst    # 第二版
```

---

### 1.2 喇叭天线模型

```
年月日_hex_circle_antenna_[馈源位置]_[序号].cst
```

#### 格式说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 年月日 | 6位日期 | 260413 |
| hex_circle_antenna | 项目标识（圆形喇叭天线） | hex_circle_antenna |
| 馈源位置 | center / offcenter / center_mm | center |
| mm | 表示该文件为喇叭天线模型 | mm |
| 序号 | **不添加**（center 和 offcenter 已区分类型） | - |

#### 示例

```
260413_hex_circle_antenna_center_mm.cst     # 中心喇叭天线（直接用center区分）
260413_hex_circle_antenna_offcenter_mm.cst  # 偏心喇叭天线
```

## 二、仿真结果命名

### 2.1 近场/远场结果

```
年月日_hex_[阵面尺寸]_[功能]_[馈源位置]_[轴向/位置]_序号.txt
```

#### 格式说明

| 字段 | 说明 | 示例 |
|------|------|------|
| 轴向/位置 | z距离或切割位置 | z-5, z100, x50_y0 |
| 序号 | **仅当同一日期同一轴向有多个时才加** | 01, 02 |

#### 示例

```
260416_hex_180_180_colli_center_z-5.txt    # z=-5mm处近场
260416_hex_180_180_colli_center_z100.txt  # z=100mm处近场
260416_hex_180_180_steer_offcenter_x50.txt # x=50mm处切割面
```

---

### 2.2 结果文件夹

与txt同名，添加 `_data` 后缀：

```
年月日_hex_[阵面尺寸]_[功能]_[馈源位置]_[轴向位置]_序号_data/
```

#### 文件夹内容

| 文件 | 说明 |
|------|------|
| Left_circular_Magnitude.jpg | 左旋圆极化幅值热图 |
| Left_circular_Phase.jpg | 左旋圆极化相位热图 |
| Right_circular_Magnitude.jpg | 右旋圆极化幅值热图 |
| Right_circular_Phase.jpg | 右旋圆极化相位热图 |
| *_mm.csv | 含坐标的矩阵数据 |
| *_nopos.csv | 无坐标的纯矩阵数据 |

---

## 三、阵面尺寸规范

| 类型 | 格式 | 示例 |
|------|------|------|
| 方形阵面 | 宽_高 | 180_180, 200_150 |
| 圆形阵面 | r直径 | r180, r200 |
| 六边形阵面 | hex_边数_对角直径 | hex_6_180 |

---

## 四、功能标识

| 标识 | 含义 |
|------|------|
| colli | 准直 (Collimated) |
| steer | 波束扫描 (Steerable) |
| multi | 多波束 (Multi-beam) |
| vortex | 涡旋波 (OAM/Vortex) |
| lcp | 左旋圆极化 (Left Circular Polarization) |
| rcp | 右旋圆极化 (Right Circular Polarization) |

---

## 五、命名检查清单

新建文件前确认：
- [ ] 日期为6位格式（YYMMDD）
- [ ] 阵面尺寸格式正确（方形用_分隔，圆形加r前缀）
- [ ] 功能标识清晰
- [ ] 馈源位置明确（center/offcenter）
- [ ] **仅同一日期同类文件有多个时才加序号**
- [ ] CST模型与结果命名一致
